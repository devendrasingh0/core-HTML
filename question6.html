1. HTML
    2. Primary Use Cases key features or benefits
    • Defining the structure of a web page.
    • Creating headings, paragraphs, lists, and other elements.
    • Embedding images, videos, and other multimedia content.
    • Easy to learn and use.
    • Cross-browser compatible.
    • Provides a foundation for CSS and JavaScript.
  2. CSS
    3. Primary Use Cases key features or benefits
    • Styling the appearance of a web page.
    • Controlling the font, color, size, and spacing of text.
    • Applying background images, borders, and shadows.
    • Creating responsive designs.
    • Allows for precise control over visual elements.
    • Encourages separation of concerns.
    • Supports responsive design.
  3. JavaScript
    4. Primary Use Cases key features or benefits
    • Adding interactivity to a web page.
    • Creating dynamic user interfaces.
    • Animating elements on a web page.
    • Manipulating data on the client side.
    • Allows for dynamic web pages.
    • Can be used to create complex user interfaces.
    • Supports a variety of libraries and frameworks.
    • Can be used to build both front-end and back-end applications.
