orderd list of HTML tags

  1. hyperlink: The tag defines a hyperlink, which is used to link from one page to another.
  2. comments: The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers.
  3. body tag: The element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  4. br tag: The
    tag inserts a single line break.
  5. body tag: The element is a container for metadata (data about data) and is placed between the tag and the tag.
  6. header tag: The
    element represents a container for introductory content or a set of navigational links.
  7. img tag: The tag is used to embed an image in an HTML page.
